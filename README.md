# Machine Learning Lab Assignments

This repository contains lab assignments from the Machine Learning course at the University of Debrecen.

## Contents

- **Lab 01** - Introduction to Machine Learning Basics
- **Lab 02** - Linear Regression with One Variable
- **Lab 03** - Linear Regression with Multiple Variables
- **Lab 04** - Logistic Regression
- **Lab 05** - Logistic Regression with Regularization
- **Lab 06** - Multiclass Classification
- **Lab 07** - Neural Network and Back Propagation
- **Lab 08** - Bias vs. Variance
- **Lab 09** - Support Vector Machine (SVM)
- **Lab 10** - K-means Clustering

## Requirements

Each lab folder may contain its own `requirements.txt` file. To install dependencies for a specific lab:

```bash
cd "Lab XX - [Lab Name]"
pip install -r requirements.txt
```

## Usage

Navigate to the specific lab folder and follow the instructions in the respective README files.
